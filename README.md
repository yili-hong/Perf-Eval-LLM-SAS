# Perf-Eval-LLM-SAS
Data Analysis for LLM SAS Code Rating

This repository contains the Python and R code used for the data analysis in the paper "Performance Evaluation of Large Language Models in Statistical Programming with SAS" by Xinyi Song, Kexin Xie, Lina Lee, Ruizhe Chen, Jared Clark, Hao He, Haoran He, Jie Min, Xinlei Zhang, Simin Zheng, Zhiyang Zhang, Xinwei Deng, and Yili Hong.

This repository also provides the individual-level rating scores. The StatLLM dataset is available in a separate repository: https://github.com/yili-hong/StatLLM.
